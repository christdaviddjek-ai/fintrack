🧪 TESTER VOTRE APPLICATION
═════════════════════════════════════════════════════════════════════════════

Test 1 - Inscription:
  1. Aller sur la page d'accueil (index.html)
  2. Cliquer sur "Créer un compte"
  3. Remplir le formulaire
  4. Cliquer sur "S'inscrire"

Test 2 - Connexion:
  1. Se connecter avec les identifiants créés
  2. Vérifier la redirection vers le tableau de bord

Test 3 - Ajouter une dépense:
  1. Cliquer sur "Ajouter Dépense"
  2. Remplir: montant, description, catégorie, date
  3. Cliquer sur "Enregistrer"

Test 4 - Voir les dépenses:
  1. Cliquer sur "Mes Dépenses"
  2. Observer la liste avec filtres

Test 5 - Modifier/Supprimer:
  1. Cliquer sur "Modifier" pour éditer
  2. Cliquer sur "Supprimer" pour effacer

Test 6 - Tableau de bord:
  1. Cliquer sur "Tableau de Bord"
  2. Voir les statistiques et graphiques

Pour plus de détails, consultez TESTING_GUIDE.md


❓ CONFIGURATION BD
═════════════════════════════════════════════════════════════════════════════

Fichier: config.php

Par défaut:
  Host:     localhost
  User:     root
  Password: (vide)
  Database: gestion_depenses

Si votre MySQL a un mot de passe, modifiez la ligne:
  $password='votre_mot_de_passe';
